FCC - Image Search Abstraction Layer

User Stories:

  1. Get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. Paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. Get a list of the most recently submitted search strings.

Example usage:

Search for 'funny cat' and skip the first 10 results:
https://fcc-324-imagesearch.herokuapp.com/search/funny%20cat?offset=10
https://fcc-324-imagesearch.herokuapp.com/search/apple
Display the most recent searches: 
https://fcc-324-imagesearch.herokuapp.com/latest
